# SCRT(系统清理和诊断工具)
## 一款自制的实用批处理程序（单文件）

<hr>

### 软件简介
#### 本程序能够测试网络连通性、安全又干净地清理电脑产生的临时文件，使用sfc和dism来修复系统文件，使用mrt清除病毒等
### 使用须知
#### 本程序仅支持Windows系统，仅能在Windows10及更高版本的Windows系统上运行（Windows8未经测试，但理论上可行）
